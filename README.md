## Краулер для выкачки текстовых страниц (ОИП, задание 1)

**Студент:** Станчу Данил Владимирович, группа 11-208

## Запуск

Для запуска необходимо выполнить следующую команду в корне репозитория: 

```bash
dotnet run --project src/Crawler.csproj 
```

## Краулер

- Читает URL из `urls.json`  

- Выкачивает 100 страниц  

- Сохраняет каждую страницу в папку `pages/` как `1.txt`, `2.txt`, ...  

- Создаёт в корне проекта файл index.txt в формате: `номер_файла.txt`  

## Результат

- `pages/` - полученные страницы (1.txt, 2.txt, ...). 

- `index.txt` — соответствие номера файла и ссылки
